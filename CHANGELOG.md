# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Created GitHub Pages website in `github-pages-website/` directory
- Added static HTML/CSS website with modern, responsive design
- Added privacy policy page (`privacy.html`)
- Added comprehensive CSS styling with semantic color variables
- Added `.nojekyll` file to ensure proper GitHub Pages deployment
- Added `robots.txt` for search engine optimization
- Added Open Graph meta tags for better social media sharing
- Added skip-to-content link for improved accessibility
- Implemented hero banner section with banner image
- Added logo image in navigation header
- Created highlight sections showcasing repository benefits

- Reduced banner image max-height from 400px to 250px for better visual balance
- Updated gradient backgrounds to match new blue theme
- Standardized HTML formatting across all pages (consistent indentation and structure)

### Fixed
- Fixed HTML formatting inconsistencies in privacy.html
- Removed erroneous asterisks from logo text in navigation
- Ensured all paths use absolute paths for GitHub Pages compatibility
- Standardized CSS variable naming conventions
- Improved code consistency and maintainability
